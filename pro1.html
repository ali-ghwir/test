Hypertext Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.

HTML

HTML، وهي اختصار لـ HyperText Markup Language، هي لغة ترميز تُستخدم لإنشاء صفحات الويب وتحديد بنيتها ومحتواها. تُعتبر HTML الأساس الذي تقوم عليه جميع مواقع الويب، حيث تُستخدم لتحديد العناصر المختلفة للصفحة مثل النصوص، الصور، الروابط، القوائم، الجداول، النماذج وغيرها.

css

CSS، وهي اختصار لـ Cascading Style Sheets، هي لغة تصميم تُستخدم مع HTML لتنسيق صفحات الويب من حيث المظهر والشكل. باستخدام CSS، يمكنك التحكم في ألوان النصوص، أحجام الخطوط، ترتيب العناصر، الخلفيات، الحدود، والعديد من جوانب التصميم الأخرى.

JavaScript

JavaScript (JS) هي لغة برمجة قوية تُستخدم في تطوير الويب لإضافة التفاعل والديناميكية إلى صفحات المواقع. بينما تُستخدم HTML لبناء الهيكل وCSS للتصميم، تأتي JavaScript لتضيف الحركة والتفاعل مما يجعل المواقع أكثر ج اذبية واستجابةً للمستخدمين.
